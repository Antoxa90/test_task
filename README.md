<h2>Start</h2>

<code>npm install</code>

Configurate mysql server data:

<code>
  let con = mysql.createConnection({
    host: "localhost",
    user: "yourusername",
    password: "yourpassword"
  });
</code>

There are test values to populate the table.

Start node.js server:

<code>npm start</code>

Open http://localhost:3000
